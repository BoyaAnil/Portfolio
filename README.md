# ANIL BOYA Portfolio

Professional corporate portfolio website for **ANIL BOYA** (Full Stack Developer), built with pure HTML, CSS, and JavaScript.

## Links

- Live Site: <https://anilboyaportfolio.netlify.app/>

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Features

- Fully responsive layout (mobile, tablet, desktop)
- Dark-first UI with theme toggle and localStorage persistence
- Sticky navigation with active section highlight on scroll
- Smooth anchor scrolling
- Scroll reveal animations using `IntersectionObserver`
- Project cards with category filters (All / Python / Web / AI)
- Contact section with click-to-copy email
- SEO basics (title, meta description, OpenGraph tags, favicon placeholder)
